# Kanban 项目管理可视化工具

面向敏捷开发团队的项目管理可视化工具，帮助管理者监控和管理每周一次的迭代进度。

## 项目目标

- **输入**: 结构化数据文本（JSON）
- **输出**: 网页可视化仪表盘
- **核心价值**: 实时监控迭代进度、资源分配优化、团队协作效率提升

## 技术栈

- **前端**: 原生 HTML5 + CSS3 + ES6+
- **可视化**: ECharts / Chart.js
- **数据**: JSON 结构化数据

## 项目状态

当前阶段: **MVP 0 - 基础架构搭建**
